Exponential Growth or Decay is when a variable is increasing or decreasing an exponential rate (not a straight line). We usually see Exponential Growth and Decay problems relating to populations, bacteria, temperature, etc, usually dealing with time. It turns out that if a function is exponential, as many applications are, the rate of change of a variable is proportional to the value of that variable. We use a basic Function in order to get net change.
